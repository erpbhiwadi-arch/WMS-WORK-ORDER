Multi Wire Drawing (MWD) WORK ORDER AND WMS PROCESS

WORK ORDER PROCESS

How to open work order page.

  • Transaction
  • Manufacturing
  • Enter Work Order

WORK ORDER STEPS

How to fill details step by step.

  • STEP-1 --> SUBSIDIARY --> MALHOTRA CABLES PRIVATE LIMITED
  • STEP-2 --> ASSEMBLY --> MWA-16/0.21
  • STEP-3 --> DEPARTMENT --> PRODUCTION - WIRE DRAWING
  • STEP-4 --> CLASS --> MWD
  • STEP-5 --> LOCATION --> BHIWADI
  • STEP-6 --> BILL OF MATERIAL --> MWA-16/0.21
  • STEP-7 --> BILL OF MATERIAL REISION --> MWA-16/0.21
  • STEP-8 --> QUANTITY --> DECIDED QUANTITY

    • STEP-9 --> -->SAVE

RBD-WMS STEPS

How to Open WMS Mobile APP.

  • STEP-1 --> SETUP
  • STEP-2 --> CUSTOM
  • STEP-3 --> MOBILE APP

RBD-WMS STEPS

How to WORK ON WMS FOR RBD.

  • STEP-1 --> WMS
  • STEP-2 --> MCPL WORK ORDER
  • STEP-3 --> PICKING
  • STEP-4 --> SELECT WORK ORDER
  • STEP-5 --> PICK ITEM
  • STEP-6 --> ENTER ITEM
  • STEP-7 --> ENTER QUANTITY & SCAN LOT
  • STEP-8 --> NEXT WORK ORDER

RBD-WMS STEPS

How to WORK ON WMS FOR RBD.

  • STEP-1 --> CLICK ON THREE BARS
  • STEP-2 --> GO TO MAIN MENU
  • STEP-3 --> MCPL WORK ORDER NUMBER
  • STEP-4 --> BUILD ASSEMBLY
  • STEP-5 --> ENTER WORK ORDER
  • STEP-6 --> FILL QUANTITY/ CLICK ON ENTER QUANTITY / AUTO GENERATE LOT NUMBER
  • STEP-7 --> ENTER EXPIRY DATE/ CLICK ON PRINT/ SELECT MWD IN TEMPLATE.
  • NOTE (DATE SHOULD BE ABOVE THAN ONE YEAR)
  • STEP-8 --> ENTER LOT
